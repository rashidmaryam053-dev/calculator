# Qt Calculator

This is a **simple calculator application** made using **C++ and Qt**.
It allows users to perform basic mathematical operations using a graphical interface.

---

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Clear button to reset values
* Error message for division by zero

---

## Tools & Technologies

* C++
* Qt Framework
* Qt Creator

---

## Project Files

```
main.cpp          - Starts the application  
mainwindow.h      - Main window class  
mainwindow.cpp    - Calculator logic  
mainwindow.ui     - User interface design  
```
